# CoolAgent Design References

This folder stores design-source material used to guide the mobile MVP.

## Current Source

- `CoolAgentDesign/`: Claude Design prototype and canvas exports.

## Usage Rule

Treat these files as visual reference only. The production mobile app lives in
`frontend/` and should be implemented with Expo, React Native, TypeScript,
shared theme tokens, reusable components, mock data, and backend adapters.

Do not add mobile flows for RAG PDF/manual upload here. RAG ingestion remains a
backend/admin concern outside the mobile MVP.
